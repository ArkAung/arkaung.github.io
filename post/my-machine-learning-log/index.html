My Machine Learning Log

The purpose of this blog (or a log) is to help people who would like to start Machine Learning on his or her own. This article does not reflect the best way out there to start Machine Learning but this is how I started.

If you are unfamiliar with Machine Learning, I recommend starting out your journey with Andrew Ng’s Machine Learning course on Coursera. This course not only gives you the idea of what Machine Learning is but also covers some Mathematical Foundations of Machine Learning. Professor Ng introduces Mathematical concepts in a way that a beginner can easily ingest. But be warned that once you have finished this course, there is a ton of more Math to deal with if you really want to learn the essence of Machine Learning. Be prepared to freshen up your Linear Algebra, Probability and Multivariate Calculus to understand the intricacies of Machine Learning.

You need to have a good foundations in Mathematics to truly understand why the learning works in Machine Learning, how gradient descent actually works, why generalization actually works, how PAC learning works and so on. You don’t need to understand everything about them to implement Logistics Regression with GraphLab or any other framework out there. But if you want to build a good foundation in Machine Learning, sharpen up your Math skill and learn all the underlying concepts. The reason I am saying all those stuffs is not because I want to scare you off but because I am struggling with some complex Mathematical concepts when I really got down to the level to understand the true building blocks of Machine Learning. Remember, this is the stage where you can get discouraged and may decided to stop pursuing.

Learning from Data from CalTech is more focused on theoretical underpinnings of Machine Learning. This course is a vanilla course offered at CalTech so don’t expect it to be easy. It is going to be challenging and you have to refer from various books (two of which I will describe below) to understand the concept. You may have to go over a single lecture again and again just to make sure that you have truly understood it.

Two of the books that I would like to recommend are Understanding Machine Learning: From Theory to Algorithms by Shai Shalev-Shwartz and Shai Ben-David and Introduction to Machine Learning by Ethem Alpaydin. These may not be the best books for you but you can ask for good recommendations from folks from /r/MachineLearning. MachineLearning subreddit is a very good source to get you familiar with the world of Machine Learning as well. Check it out daily. The twitter account for that subreddit is @slashML

If you feel like you are so bogged down by all the Math stuff and want to have a break, do Machine Learning Foundations: A Case Study Approach from Coursera. This course is very easy and focuses more on implementation part, using a framework called GraphLab. GraphLab is free for academic use so it may only be useful for your educational purposes. You can build many Machine Learning applications in a very short time using GraphLab and it will be a fun journey for you to do this course. But remember, don’t get disillusioned that ML is only that.

If you are starting to venture out to Neural Networks after you have finished your Professor Ng’s course, you have to take Neural Networks for Machine Learning by one and only Professor Hinton. Geoffrey Hinton is one of the brightest luminaries in the field of Machine Learning and Neural Networks. Be warned that this course is quite a bit more Math oriented as well, compared to Professor Ng’s course.

I would recommend reading this book, Deep Learning Book. This book focuses more on Deep Neural Networks.

This is a gentle but effective free online book on Neural Networks and Deep Learning. This book is both principle-oriented and hands-on. I highly recommend this.

As I am super stoked about Self-Driving Cars, I am doing 6.S094: Deep Learning for Self-Driving Cars from MIT. The reason I take this course is that it promises that I can implement all the stuff in the course in browser. I don’t want to get any Amazon instances and burn a hole in my wallet.

I am working on a very interesting project using Deep Neural Networks. I will publish my work after this academic year.

As I am doing my Master’s in Computer Science at Worcester Polytechnic Institute, I am taking Machine Learning by Professor Ruiz and Deep Neural Networks by Professor Whitehill this semester.

I will update this log from time to time.
